### Requirements

  * PHP ^7.3.0
  * Extensions: mbstring, curl, dom, xml,zip, sqlite3
  * Composer
  * Node.js & npm
  * SQLite for local
  * [heroku cli](https://devcenter.heroku.com/articles/heroku-cli#download-and-install)


### Setup

1. clone repository
2. > make setup


### Psql DB

```
DB_CONNECTION=pgsql
DB_DATABASE=seo_analyzer
DB_USERNAME=username
DB_PASSWORD=passwd
```
> make resetPsqlDB

### Sqlite DB

```
DB_CONNECTION=sqlite
```
> make resetSqliteDB
